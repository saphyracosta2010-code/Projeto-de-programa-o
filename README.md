#  Sistema de Biblioteca em Python

Este projeto é um **sistema de biblioteca simples em Python**.  
Ele permite que você liste livros, adicione novos, remova existentes,
busque por gênero e marque livros como favoritos.

---

## Funcionalidades
✔️ Listar todos os livros  
✔️ Cadastrar novos livros  
✔️ Remover livros  
✔️ Buscar livros por gênero  
✔️ Adicionar livros aos favoritos  
✔️ Mostrar livros favoritos

---

## Tecnologias
Esse projeto foi feito com:
-  Python 3 (sem outras dependências externas)
